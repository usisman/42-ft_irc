# ft_irc

A TCP-based IRC server implementation written in C++ (C++98).

## 🚀 Features
- Multi-client support
- Channel system
- Message broadcasting
- Basic IRC command handling

## 🛠️ Technologies
- C++
- TCP/IP sockets
- Unix networking

## 🧠 What I Learned
- Socket programming and TCP communication
- Handling multiple clients
- Network protocols and server architecture
